About

The KObject Kernel is a fun work-in-progress operating system kernel that we are working on for fun. KobjKern uses Limine as its bootloader, and supports multiple architectures.

The specific architectures that are supported are x86_64, aarch64, and riscv64. Currently we are working on getting KobjKern to compile and boot.
